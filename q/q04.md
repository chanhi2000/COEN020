# q04

Translate each of the following into a minimal sequence of ARM instructions, where the variables are declared as:

```c
int8_t	s8,	*ps8;
int16_t	s16;
int32_t	s32;
int64_t	s64;
```

## 1.
```c
s32 = s16;
```


## 2.
```c
ps8 = &s8;
```


## 3.
```c
s16 = s32 / 5;
```


## 4.
```c
s64 = s64 - 5
```
