## 2.
Convert the decimal number $$17.5_{10}$$ to an equivalent base 6 representation.

#### 2.
- WHOLE NUMBER

| operation | quotient | remainder |
| :-------: | :------: | :-------: |
| $$17\div6$$ | $$2$$ | $$5$$ |

- FRACTIONAL NUMBER

| operation | fractional part | whole part |
| :-------: | :-------------: | :--------: |
| $$0.5\times6$$ | $$0$$ | $$3$$ |
| $$0.0\times6$$ | $$0$$ | $$0$$ |
| $$\vdots$$ | $$\vdots$$ | $$\vdots$$ |

$$\therefore\:17.5_{10}=25.3_{6}$$
